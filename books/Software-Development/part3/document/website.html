Software Development

Github 搭建主页

1. 安装 Ruby

sudo apt-get install ruby

2. 安装 jekyll

sudo apt install jekyll

3. 选择自己喜欢的主题

主题可以在这里下载压缩包并解压，http://jekyllthemes.org/
解压后就打开浏览器本地测试了，http://127.0.0.1:4000/ 

jekyll serve

提示缺什么包就补上什么包

sudo gem install github-pages
sudo gem install jekyll-sitemap

端口可以在 _config.yml 里修改

host:        0.0.0.0
port:        80

如果修改主页需要重新编译

jekyll build

4. 提交到 github

在 github 上创建项目命名为yourname.github.io，在 Settings - Options - Github Pages 就可以看到发布的主页了。
如果想要修改域名，创建 CNAME 文件，里面存入域名就可以了，比如:

wuhanstudio.cn
